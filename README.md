# FFTOcean 

## 简介  

快速离散傅里叶变换实现的实时水面模拟。使用海洋学统计模型。

## 效果 (暂时) 

![image](https://github.com/kevin20x2/FFTOcean/blob/master/GIF2.gif?raw=true) 

## 已实现  

* Mesh 的生成  
* 频谱的初始化  
* 基于GPU的快速傅里叶变换
* 由水平偏移得到的法线
* 基本渲染、雾效
* 基础白浪
